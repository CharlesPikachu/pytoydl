# Install

#### Pip install
```
run "pip install pytoydl"
```

#### Source code install
```sh
(1) Offline
Step1: git clone https://github.com/CharlesPikachu/pytoydl.git
Step2: cd pytoydl -> run "python setup.py install"
(2) Online
run "pip install git+https://github.com/CharlesPikachu/pytoydl.git@master"
```
